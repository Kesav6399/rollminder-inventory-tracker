# RollMinder – Paper Roll Inventory & Reminder System

## Overview
RollMinder is an operational workflow tracking application designed for managing paper roll supply, client records, product usage, orders, and reminder-based delivery follow-ups.

The project focuses on solving a real business problem where paper roll suppliers need to track client consumption, expected reorder timelines, and reminders before stock runs out.

---

## Problem Statement
Businesses supplying paper rolls to clients often face challenges such as:
- Tracking multiple clients and products manually
- Remembering delivery/reorder dates
- Monitoring product usage cycles
- Managing client-wise product history
- Reducing missed follow-ups

---

## Solution
RollMinder provides a structured application workflow to manage:
- Client details
- Product information
- Client-product mapping
- Orders and billing records
- Reminder scheduling
- Dashboard-based operational overview

## System Workflow
1. Add client details
2. Add product details
3. Map client with product
4. Create order record
5. Set delivery/reminder cycle
6. Track upcoming reminders
7. View operational summary
